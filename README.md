# nlp-proj-LPD
 
 Hello,
The goal of this project was to identify LPD possible in sentences for hebrew

for example:
"היה לי לוחם ממש טוב" <=> "היה לי חלום ממש טוב",
"רוצה לבוא לאכול גדילה" <=> "רוצה לבוא לאכול גלידה"
